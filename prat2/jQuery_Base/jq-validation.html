Form Validation (JavaScript)

Back to index

Validating forms on the client side using jQuery validation plugin.

  • Consider the following rules:
    1. The field name is mandatory
    2. The field grade must be an integer value between 0 and 20 (inclusive)
    3. The field evaluation period is required
  • The validation is performed in two moments:
    • Whenever the value changes from a valid value to an invalid one
    • Before submitting the form data
  • Data can only be submitted if the values of both fields are valid. When the form is submitted the page should show an alert stating "Form submitted".

Note: Use a separate JavaScript code file "validation.js"
