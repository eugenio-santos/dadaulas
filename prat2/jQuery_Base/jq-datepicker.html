Datepicker (jQuery UI)

Back to index

Displays four datepickers using the jQuery UI Widgets.

Two of the datepickers use the portuguese language. The other two are in english.

For each language there will be a version without any configuration and a version that allows only dates of the last month (from one month ago until today). Also, this last version changes the date format to "day-month-year".

Use the redmond theme. You must change the HTML file to load the additional required JavaScript and CSS files

Note: Use a separate JavaScript code file: datepicker.js
