Slideshow 2 (jQuery)

Back to index

Displays an image slide show using the Slick plugin.

Consider the following rules:

  • show dots;
  • autoplay (2s between each transition). Pauses autoplay when the mouse is hover a dot;
  • shows 5 slides and scrolls one slide on each transition;
  • support the following responsive breakpoints:
    • <=980px – shows 4 slides;
    • <=768px – shows 3 slides;
    • <=568px – shows only one slide and don't show dots;

Note: Use a separate JavaScript code file "slideshow-2.js"
