Use the undocumented webservice

http://www.omdbapi.com/?s=search+string
to search for movies.

Replace "search+string" by the string you want to query.

The returned object has a property "Search" with an array of objects. These objects have the following properties:

  • imdbID
  • Title
  • Year
  • Type
  • Poster

Note: Use a separate JavaScript code file "imdb.js"
