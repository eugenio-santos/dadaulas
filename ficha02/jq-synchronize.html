Synchronize Fields (jQuery)

Back to index

Synchronize field values of a form with text in a different section of the page.

The text of the elements of the "top" section, display the value of the fields in the form.

Note that the behavior of the fields Name and Grade are different because of the event used in each field. In the field "Name" it is used the event onchange and in "Grade" the event onkeyup . On load the "top" section should show the default values on each field. When pressing reset all "text" fields are cleared and the select resets to the first option.

Note: Use a separate JavaScript code file "synchronize.js"
