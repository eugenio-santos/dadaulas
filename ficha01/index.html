ECMAScript 5 (JavaScript)

Mandatory Exercises

  1. Hello World Write "Hello World!" using JavaScript.
  2. Alert box Show text in alert box.
  3. Show and hide sections Shows and hides a section of a page when a user clicks or passes over a certain HTML element.
  4. Synchronize Fields Synchronize field values of a form with text in a different section of the page.
  5. SlideShow
    Displays an image slide show with navigation links and automatically every 2 seconds.

Extra Exercises

  1. Form Validation Validating forms on the client side.
  2. Todo List Simple to-do list application.
